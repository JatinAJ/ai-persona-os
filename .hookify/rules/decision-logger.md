# Decision Logger

**Trigger:** End of any conversation where decisions were made

**When to use:** At the end of each session, review what was decided and log it.

**Format:**

```markdown
### [Decision Title]
**Date:** YYYY-MM-DD
**Decision:** What was decided
**Rationale:** Why this was chosen
**Alternatives considered:** Other options that were discussed
**Status:** Implemented | Pending | Deferred
**Files changed:** List of files modified/created
```

**Where to log:** `memory/decisions/2026-03.md` under today's date section.

**Examples of decisions to log:**
- Installed new skill/plugin
- Changed configuration
- Created new file or directory structure
- Chose one approach over another
- Deferred or rejected an option
- Set up new automation or hook
